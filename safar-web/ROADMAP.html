📅 1.5 oy 🌍 5 shahar 🗣️ 3 til (EN · UZ · RU) 💻 Web 🛠️ Admin Panel 📱 Mobile (iOS + Android) 📍 GPS auto-play 🗺️ Mapbox 💳 Uzum Merchant 🏨 Booking.com 🖥️ VPS (airnet.uz) ☁️ Cloudflare CDN

1Loyiha haqida

Safar — O'zbekistondagi 5 ta Buyuk Ipak Yo'li shahri (Toshkent, Samarqand, Surxondaryo, Buxoro, Xiva) uchun premium darajadagi raqamli muzey gid platformasi. Xalqaro turistlarga tarixiy obidalar haqida professional ma'lumot, GPS bilan avtomatik audio playback, va madaniy tajriba beradi.

MVP maqsadi: 1.5 oyda mijozga 3 ta ishlaydigan platformani topshirish — Web + Admin Panel + Mobile (iOS/Android). Mijoz o'zi content (matn, audio, rasm) qo'shadigan dinamik tizim — kodga tegmasdan admin paneldan barcha narsani boshqarish imkoniyati.

8Asosiy printsiplar

1. Backend = oltin manba. Sifatli, type-safe backend bo'lsa — har yangi feature web + mobile'ga 1 kunda yetkaziladi (Zod shared schema + TanStack Query).

2. Hech narsa hardcoded emas — barcha kontent, til, narx, kategoriya admin paneldan boshqariladi.

3. Type-safe top to bottom — packages/shared'da Zod schema'lar, backend va frontend va mobile bir xil tipni ko'radi.

4. Mobile-first — turistlar telefondan kiradi. Mobile app real GPS bilan field-test qilinadi.

5. Premium estetika — Mapbox Studio'da custom Islamic-Silk Road style, shadcn/ui (web), NativeWind (mobile).

6. Content-status workflow — AI draft → human review → publish. AI content tasodifan public ko'rinmaydi.

7. Mijoz mustaqilligi — content (matn, audio, rasm) to'liq mijoz mas'uliyati. Admin panel orqali to'ldiriladi.
